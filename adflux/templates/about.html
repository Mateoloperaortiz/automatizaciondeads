Acerca de AdFlux

¿Qué es AdFlux?

AdFlux es una plataforma de automatización de publicidad para ofertas de empleo, diseñada para Magneto365, una empresa colombiana de búsqueda de empleo. La plataforma permite la publicación automatizada de anuncios de empleo en múltiples plataformas de redes sociales, incluyendo Meta (Facebook/Instagram), X, Google, TikTok y Snapchat.

Características principales

  • Integración con Meta Ads API y Google Ads API para crear y gestionar campañas publicitarias
  • Segmentación de candidatos mediante algoritmos de aprendizaje automático (K-means)
  • Generación de contenido creativo para anuncios utilizando la API de Google Gemini
  • Interfaz web intuitiva y comandos CLI para gestionar ofertas de empleo, candidatos y campañas publicitarias
  • Procesamiento en segundo plano con Celery y Redis para tareas como la creación de campañas y sincronización de datos

Tecnologías utilizadas

  • Backend: Flask (Python)
  • Base de datos: SQLAlchemy ORM
  • Frontend: Plantillas Flask con Jinja2 y Tailwind CSS
  • Procesamiento en segundo plano: Celery con Redis
  • APIs externas: Meta Ads API, Google Ads API, Google Gemini API
  • Aprendizaje automático: Algoritmo K-means para segmentación

Equipo de desarrollo

AdFlux es un proyecto universitario desarrollado por un equipo de estudiantes de ingeniería de software.
